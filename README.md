# Site — Marcela Tupinambá

Site institucional simples e elegante para a nutricionista Marcela Tupinambá.

## Como visualizar

Abra o arquivo `index.html` no navegador ou use um servidor local:

```bash
npx serve .
```

## Personalização

Edite os seguintes itens em `index.html`:

- Número do WhatsApp (links `wa.me`)
- E-mail e Instagram
- Número do CRN
- Foto profissional (substitua o placeholder na seção "Sobre")

Para adicionar a foto, troque o bloco `.image-placeholder` por:

```html
<img src="foto.jpg" alt="Marcela Tupinambá, nutricionista">
```
